Project Summary

This project is a task management application built to help users efficiently organize and manage their tasks. The app allows users to add, update, mark as completed, and delete tasks. It includes features like task categorization, priority setting, due date tracking, and filtering tasks based on their status (e.g., pending, completed, overdue). The application is built using Swift and Core Data for storage.

Project Vision

The vision of this project is to provide an intuitive and user-friendly platform where users can manage their personal tasks with ease. By offering features such as task prioritization, task completion tracking, and notifications for pending or overdue tasks, the app aims to improve task productivity and organization. The app will cater to both personal and professional task management needs.

  • Task Entity: A Core Data entity represents the task model, containing attributes like title, priority, due date, and status.

Project/Business Requirements

  • Task Management: Users should be able to add, update, and delete tasks.
  • Task Prioritization: Tasks can have priorities (e.g., Low, Medium, High) that affect their sorting.
  • Due Date Tracking: Tasks should have a due date, and users should be notified if a task is approaching its due date or overdue.
  • Status Tracking: Tasks can be marked as pending, completed, or overdue.
  • User Interface: The app needs to be simple, responsive, and easy to navigate.
  • Data Storage: Core Data should be used for persistent storage of tasks.
